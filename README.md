
# 💰 Expense Tracker (React + Django)

A full-stack Expense Tracker web application built using React (frontend) and Django (backend).  
Users can add, view, and delete expenses with a clean and responsive UI.

---

## 🚀 Features

- ➕ Add Expense
- 📋 View Expense List
- ❌ Delete Expense
- 📱 Responsive Design (Mobile + Desktop)
- 🔗 REST API integration (Django)

---

## 🛠 Tech Stack

- Frontend: React.js
- Backend: Django + Django REST Framework
- Database: SQLite
- Styling: CSS
